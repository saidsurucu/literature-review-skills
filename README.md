# literature-review-skills

Claude skill(s) for academic literature review.

- **[`literature-review/`](literature-review/)** — search and read scholarly
  literature across **23 sources** (OpenAlex, Crossref, PubMed, Europe PMC,
  Semantic Scholar, arXiv, bioRxiv/medRxiv, DOAJ, DataCite, HAL, Open Library,
  Google Books, CORE, Unpaywall, OpenCitations, Google Scholar, Emerald, Brill,
  Taylor & Francis journals + books, Wiley, Scopus, Web of Science) via free open
  APIs and the user's own Chrome. See its [README](literature-review/README.md) and
  [`SKILL.md`](literature-review/SKILL.md).
